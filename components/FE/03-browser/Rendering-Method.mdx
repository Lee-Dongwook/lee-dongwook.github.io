# Rendering Method (렌더링 방식)

### Single Page Application(SPA) 

- 한 개의 페이지로 이뤄진 애플리케이션
- 페이지의 리로딩이 없어 자연스러운 화면 전환과 좋은 사용자 경험을 제공

***

### Multi Page Application(MPA) 

- 서버로부터 새로운 페이지를 요청하고 파싱하는 방식

***

### Client Side Rendering (CSR)

- 클라이언트 영역에서 화면을 제어하는 렌더링 방식

***

### Server Side Rendering (SSR)

- 서버에서 사용자에게 보일 페이지를 모두 구성한 후 사용자에게 보여주는 방식

***
