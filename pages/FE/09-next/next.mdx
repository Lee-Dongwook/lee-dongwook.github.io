# Next.JS

React 서버 사이드 렌더링 프레임워크

*** 

### Next.JS에서 제공하는 주요 기능

#### 1. Hot Reloading

 Next.JS에서는 개발 중인 코드를 자동으로 새로고침한다.


#### 2. Automatic Routing

 `pages/` 내부 파일이 해당 이름으로 라우팅 된다.


#### 3. Single File Components

 `style jsx`를 사용하여 컴포넌트 내부에 해당 컴포넌트만의 CSS를 만들 수 있다.

#### 4. SSR (Server Side Rendering) / SSG (Static Site Generation) 지원

페이지가 서버에서 렌더링 되기 때문에 초기 로딩 속도가 빠르다.   
또한, 검색 엔진이 쉽게 크롤링하고 인덱싱할 수 있어 SEO가 향상된다.
