# JSON (JavaScript Object Notation)

```json
{
    "name" : "TripPlannerZ",
    "description": "여행 동행자 찾기 및 여행 최단 경로 제공 서비스",
    "member": {
        "total": 3,
        "FE": 1,
        "BE": 2,
    },

    "skills": {
        "FE": ["React", "TypeScript", "styled-components", "pnpm", "redux", "react-query", "Vite", "ESLint", "Prettier","Jest", "Playwright"],
        "BE": ["SpringBoot", "Java"],
        "DB": ["PostgreSQL", "Redis"],
        "DevOps": ["Github Action", "Docker", "AWS EC2"]
           
    }
}
```
