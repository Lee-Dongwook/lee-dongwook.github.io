### Performance

웹 페이지가 로드될 때의 성능을 측정한다.

성능을 평가하는 지표는 크게 6가지이다.

> **FCP (First Contentful Paint)** : 사용자가 특정 웹 페이지로 이동 시, 브라우저가 첫 번째 DOM의 콘텐츠를 렌더링하는 데 걸리는 시간을 측정한 지표

> **SI (Speed Index)** : 웹 페이지에 보이는 영역의 로딩 속도를 측정한 지표

> **LCP (Largest Contentful Paint)** : 웹 페이지의 뷰포트에서 가장 큰 영역을 차지하는 이미지나 텍스트 블록의 렌더링 시간을 측정한 지표 

> **TTI (Time to Interactive)** : 웹 페이지가 완전하게 상호작용할 수 있는 상태가 될 때까지 걸리는 시간을 측정한 지표

> **TBT (Total Blocking Time)** : 웹 페이지가 사용자 입력에 응답하지 못하도록 차단된 총 시간

> **CLS (Cumulative Layout Shift)** : 불필요한 레이아웃의 변경 빈도를 측정하여 시각적 안정성을 판단하는 지표
