# Cookie (쿠키)

서버가 사용자의 웹 브라우저에 전송하는 작은 데이터 조각

브라우저는 해당 데이터 조각들을 저장하였다가 동일한 서버에 재 요청 시 저장된 데이터를 함께 전송함

클라이언트에 저장되고 클라이언트의 메모리를 사용하기 때문에 서버 자원을 사용하지 않는다.

***

### 사용 목적

1. 세션 관리 (Session Management)
2. 개인화 (Personalization)
3. 트래킹 (Tracking)

***

### 라이프타임

1. 세션 쿠키 (Session Cookie) : 현재 세션이 끝날 때 삭제 된다.
2. 영속 쿠키 (Permenant Cookie) : `Expires` 속성에 명시된 날짜 혹은 `Max-Age` 속성에 명시된 기간 이후에 삭제 된다.
