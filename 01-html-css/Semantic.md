# Semantic

의미의, 의미론적인 이라는 단어

문서를 작성할 때, 의미에 맞는 태그를 사용하여 작성하는 방법을 `시맨틱하게 작성한다`라고 한다.

***

### 시맨틱 태그 간단 정리

1. `<h1> ~ <h6>` - 제목을 나타내는 태그

2. `<header>` - 제목이나 대표 이미지가 들어가는 요소, 머리글 부분

3. `<footer>` - 전체 문서 또는 Section의 바닥글

4. `<main>` - 페이지의 메인 콘텐츠 영역

5. `<article>` - 단일 게시물의 영역을 나타냄

6. `<section>` - 단일 부분을 그룹화하여 나타냄

7. `<aside>` - 간접적인 추가 정보에 대한 요소

8. `<nav>` - 다른 페이지로의 이동이나 특정 영역 탐색 등을 수행
