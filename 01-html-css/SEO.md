# SEO (Search Engine Optimization)

검색 엔진 최적화

검색 사이트 (네이버, 구글, 다음 등등)의 상단에 노출시켜 조회수가 증가할 수 있도록 개선

***

### 검색 엔진 최적화를 위한 방법

1. HTML 문서를 시맨틱하게 작성한다.

2. `<title>` 부분을 적절하게 작성한다.

3. `<meta name="description">` 을 이용하여 페이지의 설명을 기록한다.

4. `<meta charset="UTF-8"/>` 를 사용하여 인코딩 방식을 지정한다.

5. `og(open graph)`,`twitter` 등의 프로토콜 태그를 이용하여 외부 사용자들의 유입을 증가시킨다.
